## 📚 Conteúdo Principal

### 1. 🌟 O que é HTML?
HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

### 2. 🏗️ Estrutura Básica
```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>
```
