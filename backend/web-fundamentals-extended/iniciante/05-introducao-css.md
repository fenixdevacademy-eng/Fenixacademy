## 📚 Conteúdo Principal

### 1. 🌟 Seletores CSS
```css
/* Seletor de elemento */
h1 { color: blue; }

/* Seletor de classe */
.destaque { background: yellow; }

/* Seletor de ID */
#header { font-size: 24px; }
```
