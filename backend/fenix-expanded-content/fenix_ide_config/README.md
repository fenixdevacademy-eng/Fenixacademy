## 🚀 **Como usar**

### 1️⃣ **Criar novo projeto**
```bash
# Console Application
dotnet new console -n MeuProjeto
cd MeuProjeto
dotnet run
```

### 2️⃣ **Executar código**
```bash
# Compilar e executar
dotnet build
dotnet run
```

### 3️⃣ **Adicionar dependências**
```bash
# Instalar pacote NuGet
dotnet add package Newtonsoft.Json
```

### 4️⃣ **Executar testes**
```bash
# Executar testes unitários
dotnet test
```
